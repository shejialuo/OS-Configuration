## Microsoft Todo 与四平台日历结合、同步功能

各个平台所使用的软件说明：

+ ArchLinux: 网页版
+ Windows10: 自带的日历软件
+ iPhone: 自带的日历软件以及Outlook
+ Android: Outlook

同步的原理在于ios的Microsoft Todo同步到ios自带的待办事项（添加outlook账户），通过js脚本同步到日历，进而实现四个平台的同步。
