# 1. Node

官网下载安装包，采用默认安装，安装位置为：`C:\Program Files\nodejs\`。

## 1.1 TypeScript安装

```shell
npm install -g typescript
```

## 1.2 Angular安装

```shell
npm install -g @angular/cli
```

## 1.3 Nest安装

```shell
npm i -g @nestjs/cli
```
